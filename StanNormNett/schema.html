Standardisation vs Normalisation

8 méthodes complètes — formules, exemples Python, cas d'usage & nuances

⚙️
Standardisation
centrer · réduire · z-score
📖 Définition

Transforme les données pour que chaque feature ait μ = 0 et σ = 1. Les valeurs expriment combien d'écarts-types elles s'éloignent de la moyenne. Ne borne pas les données.

FORMULE CLEF
X' = (X − μ) / σ
μ = moyenne · σ = écart-type
⟁ 4 méthodes
01 Z-Score Standardization ⌄
🔢 Formule X' = (X − moyenne) / écart-type moyenne population μ · σ écart-type (ddof=0)
📊 Résultat Moyenne ≈ 0 · Écart-type = 1
python 
import numpy as np

X = np.array([10, 20, 30])

mean = np.mean(X)
std  = np.std(X)

X_std = (X - mean) / std
print(X_std)  # [-1.225  0.  1.225]
🎯 Utilisation Cas général · Machine Learning · Méthode par défaut · Régression, PCA, SVM, LDA
⚡ Nuance Sensible aux outliers : un seul point extrême fausse μ et σ. Toujours inspecter la distribution avant.
02 Robust Standardization ⌄
🔢 Formule X' = (X − médiane) / IQR IQR = Q3 − Q1 · résistant aux outliers
📊 Pourquoi ? La moyenne est sensible aux outliers · La médiane est robuste
python 
X = np.array([2000, 3000, 5000, 1000000])

median = np.median(X)
Q1  = np.percentile(X, 25)
Q3  = np.percentile(X, 75)
IQR = Q3 - Q1

X_rob = (X - median) / IQR
print(X_rob)
🎯 Utilisation Données financières · Capteurs bruités · Données médicales · Tout jeu avec > 5% d'outliers
⚡ Nuance Ne produit pas μ=0 σ=1 au sens strict — utilise médiane et IQR. Les outliers restent mais n'influencent plus le scaling.
03 Mean Normalization Std. ⌄
🔢 Formule X' = (X − moyenne) / (max − min) Centré sur 0 · utilise le range plutôt que σ
📊 Résultat Centré autour de 0 · Valeurs entre −1 et 1 (environ)
python 
X = np.array([10, 20, 30])

mean    = np.mean(X)
min_val = np.min(X)
max_val = np.max(X)

X_mn = (X - mean) / (max_val - min_val)
print(X_mn)  # [-0.333  0.  0.333]
🎯 Utilisation Moins fréquent · Recherche académique · Modèles nécessitant un centrage sans dépendre de σ
⚡ Nuance Hybride entre standardisation et normalisation. Sensible aux outliers comme Min-Max car utilise max−min.
04 Unit Vector Standardization ⌄
🔢 Formule X' = X / ‖X‖ où ‖X‖ = √(Σ Xᵢ²) Norme euclidienne du vecteur · résultat de longueur = 1
📊 Résultat Longueur du vecteur = 1 · Direction préservée
python 
X = np.array([3, 4])

norm  = np.linalg.norm(X)
X_unit = X / norm

print(X_unit)                # [0.6 0.8]
print(np.linalg.norm(X_unit)) # 1.0
🎯 Utilisation ML avancé · NLP · Cosine similarity · Réseaux neurones · Comparaison de vecteurs de direction
⚡ Nuance Opère sur chaque ÉCHANTILLON (ligne), pas par feature (colonne). Identique à L2 Normalization — même opération, nommée différemment selon le contexte.
◈
Normalisation
bornage · mise à l'échelle · feature scaling
📖 Définition

Transforme les données dans un intervalle fixe (généralement [0,1] ou [−1,1]). Préserve les rapports proportionnels mais modifie la distribution. Idéal quand les bornes min/max sont connues.

FORMULE CLEF
X' = (X − Xmin) / (Xmax − Xmin)
résultat ∈ [0, 1] · toujours borné
⟁ 4 méthodes
01 Min-Max Normalization ⌄
🔢 Formule X' = (X − Xmin) / (Xmax − Xmin) Résultat ∈ [0, 1] · préserve les proportions
📊 Résultat Toutes les valeurs mises à l'échelle entre 0 et 1
python 
X = np.array([10, 20, 30])

X_min = np.min(X)
X_max = np.max(X)

X_norm = (X - X_min) / (X_max - X_min)
print(X_norm)  # [0.  0.5  1.]
🎯 Utilisation Réseaux neurones (sigmoid/tanh) · KNN · K-Means · Traitement d'image (pixels 0–255 → 0–1)
⚡ Nuance Très sensible aux outliers : un seul point extrême comprime toutes les autres valeurs vers 0 ou 1. Filtrer AVANT.
02 Max-Abs Normalization ⌄
🔢 Formule X' = X / max(|X|) Résultat ∈ [−1, 1] · idéal données centrées sur 0
📊 Résultat Toutes les valeurs entre −1 et 1 · Préserve le signe et le zéro
python 
X = np.array([-10, 0, 20])

max_abs = np.max(np.abs(X))

X_ma = X / max_abs
print(X_ma)  # [-0.5  0.  1.]
🎯 Utilisation Données déjà centrées sur 0 · Matrices sparses (NLP, TF-IDF) · Algorithmes ML sensibles aux valeurs absolues
⚡ Nuance Ne centre PAS les données (pas de soustraction de moyenne). Conçu pour matrices sparses — évite de les rendre denses.
03 L2 (Vector) Normalization ⌄
🔢 Formule X' = X / ‖X‖₂ où ‖X‖₂ = √(Σ Xᵢ²) Norme L2 euclidienne · très proche de Unit Vector Std.
📊 Résultat Norme du vecteur = 1 · Direction préservée
python 
X = np.array([3, 4])

norm  = np.linalg.norm(X)
X_l2  = X / norm

print(X_l2)                 # [0.6 0.8]
print(np.linalg.norm(X_l2)) # 1.0
🎯 Utilisation NLP (TF-IDF, word embeddings) · Cosine similarity · Recommandation · Clustering de documents
⚡ Nuance Opère sur chaque ÉCHANTILLON (ligne), pas par feature (colonne) — confusion très fréquente ! Identique à Unit Vector Std. mathématiquement.
04 L1 Normalization ⌄
🔢 Formule X' = X / ‖X‖₁ où ‖X‖₁ = Σ |Xᵢ| Norme L1 Manhattan · somme des valeurs absolues = 1
📊 Résultat Somme des valeurs absolues = 1 · Interprétable comme probabilité
python 
X = np.array([1, 2, 3])

norm = np.sum(np.abs(X))
X_l1 = X / norm

print(X_l1)  # [0.167 0.333 0.5]
print(np.sum(np.abs(X_l1)))  # 1.0
🎯 Utilisation Analyse de distribution · Données de fréquence · Probabilités · Histogrammes · Topic modeling
⚡ Nuance L1 produit des vecteurs plus "parcimonieux" que L2 (favorise les zéros). Choisir L1 pour la sparsité, L2 pour la similarité géométrique.

Résistant aux aberrations
NON ↓

PCA · SVM · Régression

💡 Règles d'Or

🔴
Pensez "GAUSS"
Distribution normale, PCA, SVM → Z-Score. Les résidus centrés-réduits sont la norme.
🟢
Pensez "BORNES"
Réseaux neurones, pixels d'images → MinMax. L'intervalle [0,1] doit avoir un sens métier.
🟡
Pensez "SPARSE"
NLP, TF-IDF, cosine similarity → L1 / L2 / MaxAbs. Ne pas centrer les matrices creuses !
🔵
Règle Absolue
.fit() uniquement sur le train set. Transformer train + test avec les stats du train. Jamais l'inverse.
